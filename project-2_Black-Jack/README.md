# Blackjack Game

A simple Blackjack card game built using HTML, CSS, and JavaScript.

## Features

- Start Game button
- Draw New Card
- Blackjack win detection
- Dynamic card display
- Interactive UI

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone the repository
2. Open the folder in VS Code
3. Run `index.html` in browser
